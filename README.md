# 📊 Analyse de la performance des fournisseurs – Supply Chain (Power BI)

## 🎯 Objectif du projet

Ce projet vise à analyser la performance des fournisseurs dans une chaîne d’approvisionnement à l’aide d’un modèle de scoring basé sur le délai, le coût et la qualité.

L’objectif est de :
- Identifier les fournisseurs performants à prioriser
- Détecter les fournisseurs à risque
- Comprendre les facteurs impactant la performance globale

---

## 🛠️ Outils utilisés

- Power BI  
- Power Query  
- DAX  

---

## ⚙️ Méthodologie

1. Nettoyage et préparation des données  
2. Création d’un scoring fournisseurs :
   - Délai (40%)
   - Coût (30%)
   - Qualité (30%)
3. Classification :
   - Excellent
   - Moyen
   - Risque  
4. Création du dashboard interactif  

---

## 🔍 Résultats clés

- Fournisseurs performants : Bookshop, Golf, Footwear, Outdoor, Apparel  
- Fournisseurs à risque : Technology, Health and Beauty, Fan Shop  
- Aucun pays critique à fort volume et faible performance  
- La qualité est le facteur le plus faible  
- Le coût et la qualité ne sont pas toujours corrélés  

---

## 💡 Recommandations

- Prioriser les fournisseurs performants  
- Réévaluer ou remplacer les fournisseurs à risque  
- Améliorer la qualité des produits  
- Surveiller les zones à faible performance
